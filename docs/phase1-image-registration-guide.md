## Step 1 · 작업 체크리스트 열기

체크리스트 파일: **`docs/phase1-material-images-checklist.csv`**

- A+B 그룹 153개 자재만 추출되어 있음
- 엑셀 또는 구글 시트에 그대로 import 가능 (UTF-8 BOM 포함)
- 정렬: 카테고리(시각적 중요도) → 서브카테고리 → 등급 순

### 구글 시트로 가져오기
1. 구글 시트 새 문서 생성 → **파일 → 가져오기 → 업로드**
2. CSV 파일 선택 → "스프레드시트 바꾸기" 또는 "새 시트 추가"
3. 구분 기호: **쉼표** / 텍스트를 숫자·날짜로 변환: **아니요**
4. M열(`image_url`)에 작업한 URL을 입력하며 진행

## Step 3 · 이미지 업로드 + 링크 복사

### 이미지 준비 팁
- **가로 비율 4:3 또는 1:1 권장** (자세히 모달 카드에 표시되는 비율)
- 권장 해상도: **800×600px 이상** (해상도 너무 낮으면 흐림)
- 형식: JPG/PNG/WebP 모두 OK
- 자재 이미지 출처
  - 브랜드 공식 사이트(LX하우시스, 동화자연마루, 한샘 등) 제품 이미지
  - 네이버 쇼핑 / 쿠팡 상품 사진
  - 직접 매장에서 촬영
  - 무드보드 사이트 (Pinterest, 오늘의집)

### 업로드 + 링크 복사 흐름
1. 카테고리 폴더에 이미지 업로드
2. 업로드된 파일 우클릭 → **링크 복사**
3. 클립보드의 URL을 CSV의 해당 행 `image_url` 컬럼에 붙여넣기
4. 체크 컬럼을 `☑` 로 변경

> ✅ **공유 링크 형식 무관**:
> `https://drive.google.com/file/d/{FILE_ID}/view?usp=sharing`
> `https://drive.google.com/open?id={FILE_ID}`
> `https://drive.google.com/uc?id={FILE_ID}&export=view`
> → 어느 형식이든 시스템이 자동으로 thumbnail URL로 변환합니다.

## Step 4 · CSV → materials.json 반영

작업이 어느 정도 완료되면 (예: 마루 13개 다 입력했을 때) 한 번에 반영:

### 옵션 A — 수동 편집 (소량)
구글 시트에서 material_id와 image_url을 보고 `src/data/materials.json` 의 해당 자재 객체에 직접 추가:

```json
{
  "material_id": "MAT-FL-001",
  "work_type": "flooring",
  "brand": "구정마루",
  "product_line": "구정강",
  ...,
  "image_url": "https://drive.google.com/file/d/1AbCdEf12345xyz/view?usp=sharing"
}
```

### 옵션 B — CSV 일괄 import 스크립트 (대량, 추후 추가 예정)
체크리스트 CSV를 그대로 읽어 materials.json에 일괄 병합하는 스크립트는 작업이 50개 이상 쌓이면 만들겠습니다 (필요 시 요청).

## 자주 묻는 질문

### Q. 같은 자재를 여러 곳에서 쓰는데 같은 사진을 써도 되나요?
**예.** 같은 brand × product_line 자재가 다른 work_type에 등장하더라도 같은 image_url을 재사용해도 됩니다.

### Q. 현장시공 자재 (도어/조명 일부)는 사진을 어떻게 하나요?
이건 자재가 아니라 인건비 라인이지만 사용자가 카테고리(도어/조명)는 시각화하길 원하므로,
**그 카테고리의 일반 이미지**(예: "12mm 일반 문선", "거실 간접조명")로 등록하시면 됩니다.

### Q. 이미지가 안 보여요
- 폴더가 "링크가 있는 모든 사용자" 권한인지 확인
- 일일 트래픽 한도 초과 시 일시적 차단 (1~2시간 후 회복)
- 콘솔에서 에러 확인 — 모달은 자동으로 placeholder로 fallback

### Q. 트래픽 늘면 구글 드라이브 한계가 있다고 했는데?
**MVP 단계엔 충분합니다.** 월간 활성 사용자 1만 명 정도까지는 무리 없음.
그 이상 성장하면 Vercel Blob Storage / Cloudinary로 이전 권장 (URL만 갈아끼우면 됨).

## 권장 작업 순서

1. **마루(13)** → 시각적으로 가장 인상 깊고 등급 차이 명확
2. **도배(14)** → 두 번째로 시각 효과 큼
3. **타일(21)** → 욕실 시각 결정 핵심
4. **욕실(19)** → 변기/세면대/수전 — 등급 차이 매우 큼
5. **주방가구(16)** → 등급 차이 매우 큼
6. **조명(15)** → 매그네틱·간접 등급 차이 명확
7. **도어·중문·일반가구(19)** → 현관·중문 인상 결정
8. **에어컨·실링팬(9)** → 시각 임팩트 보통
9. **창호(7)** → 보통
10. **몰딩·걸레받이·확장(20)** → 여유 시

마루부터 30~40개만 등록해도 사용자 체감 개선이 매우 큽니다.
